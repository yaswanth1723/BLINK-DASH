# 🎮 Simon Says Pro

Live Demo :- https://blink-dash-ten.vercel.app/

A modern and interactive **Simon Says memory game** built using **HTML, CSS, and JavaScript**.
Test your memory by repeating an ever-growing sequence of colors!

---

## 🚀 Features

* 🎯 Classic memory-based gameplay
* 🎨 Modern UI with glassmorphism design
* 🌙 Dark / Light theme toggle
* 🔊 Sound effects toggle
* 📳 Vibration feedback (supported devices)
* 🏆 High score tracking using localStorage
* ⚡ Smooth animations and transitions
* 🧠 Progressive difficulty (level increases each round)

---

## 🕹️ How to Play

1. Click the **Start** button to begin the game.
2. Watch the color that flashes.
3. Repeat the sequence by clicking the correct buttons.
4. Each level adds a new color to the sequence.
5. If you click the wrong color → **Game Over**.
6. Try to beat your **high score**!
